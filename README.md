# Installation
1. Before opening project rename it using "Rename.cmd".
2. Set Web as default start up project.

# Features
- Model-View-ViewModel architectural design.
- Secure default configuration.
- Easy project renaming.
- Dependency injection.
- Custom error pages.
- Code scaffolding.
- Globalization.
- Audit log.
- Site map.
- Tests.
